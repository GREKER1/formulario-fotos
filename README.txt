
# Formulario Fotos + Cloudinary + Google Sheets

## Configuración
1. Asegúrate de tener un upload preset llamado "formulario2025" en Cloudinary, marcado como unsigned.
2. La hoja en Google Sheets debe tener una hoja activa llamada "base".
3. Publica el Apps Script como Web App con permisos "Cualquiera" y método doPost().
4. Coloca la URL del Apps Script en script.js.

## Deploy en GitHub Pages
- Reemplaza los archivos en tu repositorio local con estos.
- Haz commit y push: 
    git add .
    git commit -m "Actualización funcional con CORS"
    git push
