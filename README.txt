
# Instrucciones

1. Abre index.html en tu navegador.
2. Llena el formulario, selecciona las fotos y envía.
3. Las fotos se suben a Cloudinary.
4. Los datos + URLs se mandan a tu hoja de cálculo usando Google Apps Script.
5. El Apps Script detecta la planta y envía un correo automático al contacto correspondiente.

Asegúrate de configurar bien el upload_preset en Cloudinary si cambiaste el default.
